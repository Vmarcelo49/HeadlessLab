# host-debs/ — Bundled Host Dependencies

This directory contains pre-downloaded `.deb` packages from Debian 13 (trixie) that
provide the X11 window manager and input tools required by the `headless` CLI.
They are bundled here so that LLM sandboxes / CI environments without `sudo` or
without internet access can still install everything via `bin/install-host-deps.sh`.

## Total size
~10MB (42 packages)

## Notable exclusions

- **`xvfb`** is NOT bundled. It requires kernel DRM/KMS access, so installing
  the `.deb` into `~/.local/` would not make it functional. Install it via
  `sudo apt-get install -y xvfb` on the host.
- **`Pillow`** (Python imaging) is NOT bundled as a `.deb` because the
  extracted `.deb`'s C extension may not match the user's Python version.
  Install it via `pip install --user --break-system-packages Pillow`.
- **Wine, Mesa, LLVM, bwrap** are bundled inside the `prefix/` directory
  (generated by `build-from-scratch.sh` or downloaded via `setup.sh`),
  not in `host-debs/`.

## Sources

All packages were downloaded from the Debian 13 (trixie) official repositories:

- Main pool: `http://deb.debian.org/debian/pool/main/`
- Security pool: `http://deb.debian.org/debian-security/pool/updates/`

The ImageMagick packages were downloaded from the main pool because the
security pool's `+deb13u9` versions returned HTTP 404 during packaging
(likely a transient mirror sync issue).

## Verification

To verify the integrity of these packages, you can re-download them and
compare SHA256 sums:

```bash
cd host-debs
for deb in *.deb; do
    # Extract package name and version, then re-download
    apt-get download "$(dpkg-deb -f $deb Package)"
    # Compare
    sha256sum $deb
done
```
